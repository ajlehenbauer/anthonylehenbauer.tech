I am currently a freelance/independent developer making mostly games and websites. I'd love to help you with either! Explore some of my work and have an excuse to play a game or two.

Here are some of the things I've been working on...

Shields Up

As my first solo venture into game creation, Shields Up is a new take on the classic arcade spaceship game.

The controls are z/x/c or left/down/right depending on handedness

Giraffe Game

I made this as part of a team for Global Game Jam 2020. For the timeframe, we did an amazing job with the scope and ended up with a pretty complete and enjoyable arcade platformer.

It's a great game with a controller and I don't recommend playing without one but, if you want to try it don't have a controller around, keyboard controls are E:start SPACE:jump X:catch C:throw Z:fix

Cross Analysis

My most recent project is a number puzzle game. Originally created as a prototype in Google Sheets, I am now developing the game for mobile to expand on the core gameplay.

Trivia!

This is just a simple Trivia Game using Ruby on Rails and PostgreSQL. It's not a ton of fun but it shows that I can use a framework when I need to.

Mild Interaction

Mild Interaction is a set of interactive comics I made using just some HTML, CSS, JQuery, and my poor drawing skills.
